# 📚 Cadastro de Alunos

> Sistema simples de cadastro de alunos desenvolvido para praticar conceitos de programação e organização de dados.

---

## 🚀 Sobre o Projeto

Este projeto consiste em um sistema de **gerenciamento de alunos**, permitindo realizar operações básicas como cadastro, listagem e manipulação de informações.

---

## 🛠️ Tecnologias Utilizadas

- ☕ Java
- 💻 Programação Orientada a Objetos
- 📂 Estruturas de dados (listas/arrays)
- 🛢  MySQL (Banco de Dados)

---

## ⚙️ Funcionalidades

- ✅ Cadastrar alunos  
- 📋 Listar alunos cadastrados  
- 🔍 Buscar aluno  
- ✏️ Atualizar dados  
- ❌ Remover aluno  

---

## 📸 Demonstração

Exemplo de execução:

    Digite o nome do aluno:
    > Kevin

    Digite a idade:
    > 18

    Digite a turma:
    > 1

    Aluno cadastrado com sucesso!
